def register_deriv(f, df=None)

decorator to declare the derivative df of function f

Args

f
the function to register
df
(optional) the derivative

Returns

if called as a decorator @register_deriv(f) it regsiters the decorated function as the derivative of f If called as register_deriv(f, df) it does the same. A derivative function df of a node Z has the signature df(dYdZ, argno, n.args, *n.kwargs) and returns dYdA where A is n.args[argno]
